WEBSITE REDESIGN

KILKEE THALASOTHERAPY CENTRE

This project was a website redesign for thalasotherapy centre located in Ireland. We decided to design the website in a way, that it could be significantly distinguished from tropical spa's, by completely changing colour schemes.

We have chosen warm color palette, which aim was to emphasis irish welcoming culture and mark up the place as a place everyone would love to visit.
